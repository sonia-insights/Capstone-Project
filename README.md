## Project Overview :

This capstone project focuses on analyzing Blinkit sales data to uncover business insights, understand sales performance, and support data-driven decision-making. The project demonstrates an end-to-end data analytics workflow including data cleaning, SQL analysis, Python processing, and Power BI visualization.

### Project Objectives :

-Analyze sales performance across outlets

-Identify top-performing products and categories

-Understand customer purchasing behavior

-Evaluate outlet size and location impact on sales

-Build interactive dashboards for business insights

 ### Tools & Technologies Used :
Tool	Purpose
Excel - 	Data cleaning & preprocessing
SSMS	-   Database management
Python	- Data processing & analysis
Power BI- Dashboard & visualization

 ### Dataset Description

The dataset contains Blinkit retail sales information including:

-Item Identifier

-Item Type

-Outlet Location Type

-Outlet Size

-Sales Amount

-Ratings

-Item Visibility

-Fat Content

 ### Analysis Performed:
-Data Cleaning & Transformation
-Exploratory Data Analysis (EDA)
-SQL Aggregations & Joins
-Sales Trend Analysis
-Outlet Performance Analysis
-Product Category Analysis
-Dashboard Creation in Power BI and excel

### Dashboard Features (Power BI) :
-Total Sales Overview
-Outlet-wise Performance
-Category-wise Sales Analysis
-Customer Rating Insights
-Interactive Filters & Visuals

### Skills Demonstrated

SQL Query Writing, Data Cleaning Techniques, Business Analysis, 
Data Visualization, Dashboard Development, Analytical Thinking

### How to Use
Load dataset into Microsoft SQL Server.
Execute SQL queries from sql_queries.
Run Python scripts for analysis.
Open Power BI file to explore dashboard.

 ### Key Insights
Medium-sized outlets generated consistent revenue.
Certain product categories drive majority of sales.
Customer ratings positively correlate with sales.
Urban locations outperform smaller markets.
